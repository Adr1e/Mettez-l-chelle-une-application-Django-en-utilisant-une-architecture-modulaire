Technologies utilisées
======================

Langages de programmation
-------------------------

* **Python 3.11** : Langage principal du projet
* **HTML/CSS** : Templates et styles
* **JavaScript** : Interactions côté client

Framework et bibliothèques
--------------------------

* **Django 4.2** : Framework web Python
* **Gunicorn** : Serveur WSGI pour la production
* **WhiteNoise** : Gestion des fichiers statiques en production
* **Sentry SDK** : Monitoring et suivi des erreurs
* **python-dotenv** : Gestion des variables d'environnement

Outils de développement
-----------------------

* **pytest** : Framework de tests
* **pytest-cov** : Couverture de tests
* **flake8** : Linting du code Python
* **Sphinx** : Génération de documentation

Infrastructure
--------------

* **Docker** : Conteneurisation de l'application
* **GitHub Actions** : Pipeline CI/CD
* **Docker Hub** : Registry des images Docker
* **Render** : Hébergement de l'application
* **Read the Docs** : Hébergement de la documentation
