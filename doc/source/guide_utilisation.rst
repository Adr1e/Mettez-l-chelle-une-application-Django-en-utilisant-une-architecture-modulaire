Guide d'utilisation
===================

Cas d'utilisation
-----------------

Consulter les locations
^^^^^^^^^^^^^^^^^^^^^^^

1. Accéder à la page d'accueil : http://127.0.0.1:8000/
2. Cliquer sur "Lettings"
3. La liste des locations disponibles s'affiche
4. Cliquer sur une location pour voir les détails (adresse complète)

Consulter les profils
^^^^^^^^^^^^^^^^^^^^^

1. Accéder à la page d'accueil : http://127.0.0.1:8000/
2. Cliquer sur "Profiles"
3. La liste des profils utilisateurs s'affiche
4. Cliquer sur un profil pour voir les détails (ville favorite)

Administrer le site
^^^^^^^^^^^^^^^^^^^

1. Accéder à l'interface admin : http://127.0.0.1:8000/admin/
2. Se connecter avec les identifiants administrateur
3. Gérer les utilisateurs, profils, adresses et locations

Ajouter une nouvelle location
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

1. Accéder à l'interface admin
2. Dans la section "LETTINGS", cliquer sur "Addresses" > "Add"
3. Remplir les champs de l'adresse et sauvegarder
4. Cliquer sur "Lettings" > "Add"
5. Entrer le titre et sélectionner l'adresse créée
6. Sauvegarder

Ajouter un nouveau profil
^^^^^^^^^^^^^^^^^^^^^^^^^

1. Accéder à l'interface admin
2. Dans la section "AUTHENTICATION", créer un nouvel utilisateur
3. Dans la section "PROFILES", cliquer sur "Add"
4. Sélectionner l'utilisateur et entrer sa ville favorite
5. Sauvegarder
